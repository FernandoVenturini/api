### INICIANDO PROJETO
    -> npm init -y

### INSTALANDO EXPRESS
    -> npm i express

### HTTP METODOS:
    -> GET (Listar)
    -> POST (Criar)
    -> PUT (Editar Varios)
    -> PATCH (Editar Um)
    -> DELETE (Deleta)

### RODANDO O SERVIDOR AUTOMATICAMENTE:
    -> node --watch server.js

### INSTALANDO PRISMA (FAZ A CONEXAO COM BD)
    -> npm install prisma --save-dev
